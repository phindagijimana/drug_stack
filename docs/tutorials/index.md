# Tutorials

> Six end-to-end walk-throughs from target to lead. Each one is a synthesis exercise — read the relevant chapters first, then come back.

## The six

- **[Similarity search on ChEMBL](similarity-search.md)** — turn a query molecule into a ranked hit list. The on-ramp.
- **[QSAR walkthrough](qsar-walkthrough.md)** — train a QSAR model from CSV to deployed predictor.
- **[Docking walkthrough](docking-walkthrough.md)** — receptor prep, library prep, dock, rescore, triage.
- **[Target ID from RNA-seq](target-id.md)** — DEGs to ranked candidate targets.
- **[Generative design](generative-design.md)** — REINVENT 4 against an MPO.
- **[Capstone — target to lead](capstone.md)** — the full pipeline end-to-end.

## What "end-to-end" means here

Each tutorial includes:

- A clear problem statement.
- Real public data and tools.
- Pitfalls and decision points called out.
- A reproducible script with pinned versions.
- An honest verdict at the end ("here's what worked, here's what didn't").

These are not "type these commands and get a paper" tutorials. They are how you would actually do the task on a real project.

## How to use them

- **Read the chapter sections referenced** at the top of each tutorial first.
- **Run the code yourself**, on the suggested data.
- **Modify it for your problem**. The patterns transfer; the specifics rarely do.
