# Glossary

> Short definitions for the terms used throughout this handbook. Click through for the chapter that uses each one.

**ADMET** — Absorption, Distribution, Metabolism, Excretion, Toxicity. The non-potency properties that determine whether a compound is a drug.

**Allometric scaling** — predicting human PK parameters from animal data via body-weight–scaled relationships.

**AlphaFold** — DeepMind's protein-structure-prediction model. AlphaFold 3 extends to small-molecule and ion complexes.

**Active learning** — iterative ML where the model selects the next batch to label based on uncertainty / acquisition function.

**AUROC** — area under the receiver operating characteristic; standard ML classification metric.

**Bemis-Murcko scaffold** — the central ring system + linkers of a molecule, used for scaffold-aware splits.

**Bioavailability (F)** — fraction of an oral dose reaching systemic circulation.

**Bioisostere** — a substitution that preserves biological activity while perturbing one or more physicochemical properties.

**Blood–brain barrier (BBB)** — the endothelial barrier separating blood from brain tissue.

**Caco-2** — colonic cell-line permeability assay.

**Calibration** — alignment of predicted probabilities with empirical positive rates.

**ChEMBL** — curated bioactivity database (EBI).

**Chemprop** — directed-message-passing-neural-network library for chemistry.

**Clearance (CL)** — volume of plasma cleared per unit time.

**CMap / LINCS** — Connectivity Map / Library of Integrated Network-Based Cellular Signatures.

**Conformal prediction** — distribution-free prediction-set methodology with finite-sample coverage guarantees.

**Cox proportional hazards** — survival-analysis regression.

**CYP** — cytochrome P450; major drug-metabolising enzymes.

**Docking** — predicting a ligand's pose and score within a receptor pocket.

**DUD-E** — Directory of Useful Decoys, Enhanced; benchmark for virtual screening (with known biases).

**ECFP / Morgan fingerprint** — circular-substructure binary fingerprint.

**Enamine REAL** — make-on-demand virtual-compound catalogue.

**ESM** — Meta's evolutionary-scale modeling protein language model family.

**FEP** — free-energy perturbation; relative-binding-affinity simulation method.

**FPocket** — open-source geometric pocket-detection tool.

**Generative chemistry** — ML models that propose molecules rather than score them.

**GLP** — Good Laboratory Practice; regulated lab-quality framework.

**GWAS** — genome-wide association study.

**hERG** — KCNH2 potassium channel; cardiotoxicity gatekeeper.

**Hit triage** — turning a ranked screen output into a synthesis-and-test plan.

**IC50** — concentration producing 50% inhibition.

**InChIKey** — 27-character hash of a molecule's InChI; the canonical join key.

**IND** — Investigational New Drug application; FDA permission to begin clinical trials.

**JT-VAE** — junction-tree variational autoencoder for molecular generation.

**KGNN** — knowledge-graph neural network.

**Kinase** — phosphate-transferring enzyme; major drug-target class.

**LINCS L1000** — modern Connectivity-Map–style perturbation expression dataset.

**Lipinski's rule of 5** — historical druglikeness filter for oral compounds.

**MABEL** — minimum anticipated biological effect level; first-in-human dose anchor.

**MPNN** — message-passing neural network.

**MPO** — multi-parameter optimisation; composite-objective design.

**NDA / BLA** — New Drug Application / Biologics License Application; FDA marketing-approval submission.

**Open Targets** — integrated target-disease-drug evidence platform.

**OpenMM** — open-source MD engine.

**PAINS** — Pan Assay Interference Compounds; substructure-filter set.

**PD** — pharmacodynamics.

**pIC50** — −log10(IC50 in molar); modelling-friendly potency unit.

**PK** — pharmacokinetics.

**P-glycoprotein (P-gp)** — efflux transporter; BBB and gut-wall gatekeeper.

**PoseBusters** — physical-validity checker for docking poses.

**PROTAC** — proteolysis-targeting chimera; bifunctional protein-degrader.

**QSAR** — quantitative structure–activity relationship.

**QED** — quantitative estimate of drug-likeness.

**REINVENT** — open-source RL-based generative chemistry framework.

**Repurposing** — finding new uses for existing drugs.

**RWE / RWD** — real-world evidence / data.

**SAR** — structure-activity relationship.

**Scaffold split** — train/test split that puts whole Murcko scaffolds in one side.

**SELFIES** — string molecular representation where every string is valid.

**SMARTS** — substructure query language.

**SMILES** — string molecular representation; the lingua franca.

**Synthetic accessibility (SA) score** — heuristic estimate of synthesis difficulty.

**Tanimoto** — set-intersection-over-union; the standard fingerprint similarity.

**Target validation** — establishing that modulating a target will produce the desired clinical effect.

**TPSA** — topological polar surface area.

**Tractability** — whether a molecule of some modality can engage a target sufficiently.

**Ultra-large library** — billion+ compound virtual catalogue (Enamine REAL, ZINC22).

**Vina** — AutoDock Vina; open-source docker.

**V_d** — volume of distribution.

**Virtual screening** — computationally ranking a library against a target.

**Zinc / ZINC22** — virtual compound catalogue.

**Z'** — plate-quality metric for HTS.
